# AWS Certified Developer Associate

<p style="text-align: center;">
  <img src="/images/aws-developer-associate.png" alt="AWS Cert logo" width="200">
</p>

This is a free, self-paced course to help you pass the **AWS Certified Developer Associate (DVA-C02)** exam.

The content is structured as a learning path, not a reference dump. Each section builds on the last. If you follow the order, things should click rather than pile up. That said, this course focuses on the concepts and notions you need to pass the exam — hands-on practice on your own is still key to truly getting comfortable with AWS.

## How this course is structured

The syllabus is split into **12 sections**, roughly ordered by dependency:


1. [Cloud, IAM & Encryption Foundations]({{< relref "/docs/1-cloud-iam-encryption-foundations/iam-identity-access-management" >}})
2. [Networking Essentials]({{< relref "/docs/2-networking-essentials/vpc-fundamentals/" >}})
3. [Core Compute (EC2, Lambda, ASG, ELB)]({{< relref "/docs/3-core-compute-ec2-lambda-asg-elb/ec2/" >}})
4. [Core Storage (S3, DynamoDB)]({{< relref "/docs/4-core-storage-s3-dynamodb/s3/" >}})
5. [Application Integration (SQS, SNS, EventBridge, Kinesis, Step Functions)]({{< relref "/docs/5-application-integration-sqs-sns-eventbridge-kinesis-step-functions/error-handling-retry-patterns/" >}})
6. [API Layer & Authentication (API Gateway, Cognito)]({{< relref "/docs/6-api-layer-authentication-api-gateway-cognito/api-gateway/" >}})
7. [Deployment & CI/CD (CodePipeline, SAM, CloudFormation, Beanstalk…)]({{< relref "/docs/7-deployment-cicd-codepipeline-sam-cloudformation-beanstalk/cicd-codecommit-codebuild-codedeploy-codepipeline/" >}})

The exam weights certain domains heavily — **Security (26%)** and **Deployment (24%)** together make up half the test. That's reflected in how much depth each section gets.


## Before you start

This course assumes you're comfortable with basic programming concepts and have at least a passing familiarity with cloud computing. You don't need prior AWS experience, but complete beginners may want to spend an hour on the [AWS overview](https://aws.amazon.com/what-is-aws/) first.

When you're ready, start with [IAM (Identity & Access Management) →]({{< relref "/docs/1-cloud-iam-encryption-foundations/iam-identity-access-management" >}}).
